# Website Upload Instructions

1. **Where to upload:**
   Upload ALL files and folders in this directory to the `public_html` folder of your hosting provider (or the specific subdomain folder for neofisystem.com).

2. **Files included:**
   - `index.html`: The main entry point of the website.
   - `assets/`: Contains the styling and logic (CSS/JS).
   - `api/`: Contains the PHP backend (`index.php`) and database (`db.json`).

3. **Permissions:**
   - Ensure the `api` folder has "Write" permissions (755 or 777) so the system can save new licenses to `db.json`.

That's it! Your site should work immediately after uploading.
